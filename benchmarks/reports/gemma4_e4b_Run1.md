## Category: Reasoning_Logic

### Prompt 1
**User:** Розв'яжи задачу крок за кроком: У фермера є кури та корови. Разом у них 30 голів та 100 ніг. Скільки курей і скільки корів у фермера?

**Assistant - E2E Time: 58625ms (0m 58.6s):**
Це класична задача на систему рівнянь. Розв'яжемо її крок за кроком.

### 🐓🐄 Аналіз даних:
*   **Голови (Аналогово):** Кожен тварина має по 1 голові.
*   **Ніг (Аналогово):** Кури мають по 2 ноги, Корови мають по 4 ноги.
*   **Разом:** Голів = 30; Ніг = 100.

### ⚙️ Крок 1: Складання рівнянь
Нехай:
*   $K$ – кількість курей.
*   $V$ – кількість корів.

**Рівняння 1 (заголовлі):**
$$K + V = 30$$

**Рівняння 2 (за ніг):**
$$2K + 4V = 100$$

### ⚙️ Крок 2: Розв'язання системи рівнянь
Використаємо метод підстановки або додавання (виключення).

**Метод виключення:**
Помножимо Рівняння 1 на 2, щоб коефіцієнт при $K$ у другому рівнянні зник.

$2 \cdot (K + V) = 2 \cdot 30$
$$2K + 2V = 60 \quad \text{(Нове Рівняння 3)}$$

Тепер віднімемо Нове Рівняння 3 від Рівняння 2:
$$(2K + 4V) - (2K + 2V) = 100 - 60$$
$$2V = 40$$

Знайдемо кількість корів ($V$):
$$V = 40 / 2$$
$$V = 20$$

### ⚙️ Крок 3: Знайдення кількості курей
Підставимо знайдене значення $V$ (20) у Рівняння 1:
$$K + 20 = 30$$
$$K = 30 - 20$$
$$K = 10$$

### ✅ Перевірка результату:
1. **Голови:** 10 курей + 20 корів = 30 голів. (Правильно)
2. **Ноги:** $(10 \text{ курей} \cdot 2 \text{ ноги}) + (20 \text{ корів} \cdot 4 \text{ ноги}) = 20 + 80 = 100$ ніг. (Правильно)

### 🐍 Python Код

```python
import matplotlib.pyplot as plt
import numpy as np

# 1. Дані з попередньої задачі
animal_names = ['Кури', 'Корови']
quantities = [10, 20]

# 2. Створення діаграми
plt.figure(figsize=(7, 5))  # Встановлюємо розмір фігури для кращої читабельності

plt.ylabel('Кількість (голів)', fontsize=12) # Підпис осі Y

# 4. Додавання точних значень над стовпчиками
for i in range(len(animal_names)):
    plt.text(i, quantities[i] + 1, f'{quantities[i]}', ha='center', va='bottom', fontweight='bold')

# 5. Налаштування та відображення
plt.ylim(0, max(quantities) * 1.15) # Встановлюємо ліміт Y трохи вище найбільшого значення
plt.grid(axis='y', linestyle='--', alpha=0.7) # Додаємо легку сітку для кращого читання
plt.tight_layout() # Вгамовлює елементи всередині фігури
plt.show() 
```

2.  **Дані:** Створюються два списки: `animal_names` (категорії на осі X) та `quantities` (значення на осі Y).
3.  **`plt.bar()`:** Ця функція створює стовпчикову діаграму. Ми передаємо їй назви та відповідні кількості.
4.  **`plt.title()`, `plt.xlabel()`, `plt.ylabel()`:** Ці команди додають необхідні текстові описи до діаграми, роблячи її зрозумілою.
5.  **Цикл `for` та `plt.text()`:** Це додатковий, але дуже корисний крок. Він розміщує точне числове значення (10 або 20) прямо над кожним стовпчиком, що підвищує читабельність діаграми.
